Welcome to my playground!

This application is developed and maintained by Sedat Turkoglu


In this application you will find:

  • Movies library - a project to perform CRUD opertaions by connecting to Java RESTFul API, developed using Spring REST, H2 in-memory database and JPA to communicate with the database. It also allows users to search for a movie and also sort the result. Feel free to play around with it here. Please go ahead and perform all the CRUD operations.
    The purpose of this project is to demonstrate the ability to create a working full-stack application

  • Duck Hunt - This is the classic Nintendo duck-hunt game (probably not with the same functionalities :) ) Go ahead and play with it! It is simple to play: try and click on the duck!

  • Shopping - This is an ecommerce application to allow users shop online!
